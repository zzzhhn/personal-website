## Background

Traditional university course materials are static — dense textbooks and slide decks that don't encourage active learning. This project reimagines course content as interactive single-file web applications where students can test, visualize, and explore concepts in real time.

## Platforms

| Course | Subject | Key Features |
|--------|---------|-------------|
| **FIN4210** | Corporate Finance | Knowledge graph, tiered quizzes, mock exams, TVM/NPV calculators, case studies |
| **FIN3080** | Investment Analysis | Story-driven learning, radar chart comparisons, company screener, decision trees |
| **MAT2002** | Differential Equations | Slope field simulator, real-time function graphing, step-by-step ODE solver |
| **ACT4253** | Business Ethics | Difficulty-tagged quizzes, star bookmarks, ethical framework reference |

## Technical Highlights

- **Single-file architecture**: Each platform is one self-contained HTML file (up to 22,000+ lines). No build step, no bundler — open in any browser.
- **Zero runtime dependencies**: Libraries loaded via CDN with graceful fallback. Works offline after first load.
- **Rich interactivity**: MathJax for LaTeX, Plotly/Chart.js for data visualization, custom quiz engines with progress tracking.
- **Bilingual**: Full English/Chinese toggle across finance platforms.
