## Background

Built as assignment 2 for the Tencent AI-HR bootcamp: an AI companion that helps university students track and reflect on their career growth. The core bet is that a companion is only useful if it *accumulates* — so every conversation feeds a persistent knowledge graph instead of evaporating when the chat ends.

## Growth Knowledge Graph

The signature feature is a four-layer pipeline that turns unstructured conversation into structure:

1. **Extraction** — an LLM pass pulls entities (skills, experiences, goals, people) from each conversation
2. **Merge** — new entities are deduplicated against the existing graph rather than appended blindly
3. **Clustering** — Louvain community detection groups related nodes into themes
4. **Insight** — observations are derived from graph structure (centrality, cluster growth, orphan nodes), deliberately avoiding LLM-fabricated causal claims

Resume upload (PDF / docx) parses into the same graph, so a student's documented history and their chat-discovered interests live in one connected picture, rendered as an interactive force-directed graph.
