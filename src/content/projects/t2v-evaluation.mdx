## Pipeline Overview

EvalForge unifies generative video and agent evaluation into a single config-driven pipeline. Each stage is a composable plugin — swap providers, metrics, and report formats without touching core logic.

- **Plugin Registry** — Decorator-driven `Registry[T]` pattern for providers, metrics, and reporters. Extend by registering new classes — zero core changes needed.
- **Dual LLM Backend** — Supports local open-source models (Gemma 4 via OpenAI-compatible server) and cloud APIs, with automatic fallback when local inference is unavailable.
- **Crash Recovery** — JSON checkpoint persistence with atomic writes (`tmp` + `rename`). Resume interrupted pipelines from the last completed case.
- **Async Orchestration** — `asyncio.Semaphore`-based per-provider concurrency control, N-run retry with exponential backoff, and real-time progress tracking.
- **Professional CLI** — Click-based with 7 subcommands: `run`, `generate`, `evaluate`, `classify`, `report`, `validate`, `info`.
