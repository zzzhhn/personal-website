Evaluating generative AI models is fragmented by default: T2V pipelines, agent test harnesses, and quality judges are built in isolation, making cross-model comparisons brittle and results hard to reproduce. EvalForge replaces that with a single config-driven pipeline that classifies prompts, orchestrates generation at scale, runs standardised metrics, and produces structured reports automatically — across both video generation and conversational agents.

## Engineering Highlights

- **Plugin Registry** — Decorator-driven `Registry[T]` for providers, metrics, and reporters. New integrations register via `@PROVIDER_REGISTRY.register("name")` — no core changes required.
- **Crash Recovery** — JSON checkpoint persistence with atomic writes (`tempfile` + `os.rename`). Interrupted pipelines resume from the last completed case without re-running prior generations.
- **Async Orchestration** — `asyncio.Semaphore`-based per-provider concurrency prevents rate-limit cascade failures across parallel API calls. Configurable N-run averaging with exponential-backoff retry.
- **Dual LLM Backend** — Local open-source models (Gemma 4 via OpenAI-compatible server) and cloud APIs, with automatic fallback. Configured via `base_url` + `api_key` — no code changes to switch backends.
- **Professional CLI** — Click-based with 7 subcommands: `run`, `generate`, `evaluate`, `classify`, `report`, `validate`, `info`.

## Video Evaluation Track

Based on the **VBench 2.0** protocol (arXiv 2503.21755, Mar 2025), evaluating intrinsic faithfulness across 18 dimensions in 5 categories:

- **Human Fidelity** (3 dims) — anatomy, identity, clothing consistency
- **Creativity** (2 dims) — diversity, composition novelty
- **Controllability** (7 dims) — spatial relationships, attributes, camera motion, plot, interaction
- **Physics** (4 dims) — mechanics, thermotics, material, multi-view consistency
- **Commonsense** (2 dims) — motion rationality, instance preservation

Rankings use pairwise win-rate matrices — not a single aggregate score. All scores are from the published paper (Table 2), with 4 SOTA models evaluated: Sora-480p, Kling 1.6, HunyuanVideo, CogVideoX-1.5.

## Agent Evaluation Track

500 conversations from an internal Gemma 4-based assistant evaluated through intent classification (7 categories) and 8-dimension quality scoring via LLM-as-judge with 5-shot calibration — including faithfulness to quantify hallucination rate.
