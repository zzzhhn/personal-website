Human evaluation remains the gold standard for assessing video generation quality — automated metrics miss nuances in motion realism, aesthetic coherence, and creative intent. EvalForge is a production-grade platform that structures the entire annotation workflow: admins create evaluation packages, assign items to annotators, and monitor quality through real-time analytics and anti-cheat signals.

## Annotator Workstation

The scoring interface is designed for efficiency and data quality. Annotators watch a video, then rate it across multiple configurable dimensions on a 1–5 Likert scale. Scores of 1–2 require selecting at least one failure tag — providing structured reasons alongside the rating.

- **Keyboard-driven** — Number keys (1–5) for scoring, Space for play/pause, Arrow keys for navigation, Enter to submit
- **Session persistence** — Watch progress saved to sessionStorage so partially-watched videos survive page refreshes
- **Watch ratio validation** — A progress bar shows how much of the video has been watched; submission is blocked until the configured threshold is met
- **Resizable layout** — Draggable sidebar with dimension filters and item counts

## Anti-Cheat System

Every submission passes through a multi-layer integrity pipeline, all with admin-configurable thresholds:

- **Minimum watch ratio** — Annotators must watch a configurable percentage of the video before submitting
- **Minimum dwell time** — Floor + multiplier based on video duration prevents instant submissions
- **Frequency cap** — Hourly submission limit blocks automated or careless bulk-scoring
- **Fixed-value detection** — Flags annotators whose recent scores are dominated by a single value (e.g., all 3s)
- **Low-variance detection** — Flags abnormally low standard deviation across a sliding window of recent scores

Suspicious scores are marked but still recorded — admins review flagged patterns rather than blocking annotators outright.

## Admin Dashboard

- **Annotator detail** — Per-user score distributions, completion rates, anti-cheat events, and XLSX export with Chinese font support
- **Sample browser** — Browse evaluation packages, view video assets with dimension hierarchy (parent › child)
- **Settings panel** — Live-tunable anti-cheat thresholds via slider controls, plus display toggles (hide model names per account type)
- **Analytics** — Item-level, dimension-level, and model-level score aggregation using only VALID scores

## Technical Stack

Built on **Next.js 16** App Router with React 19 Server Components. Data layer uses **Prisma 7** with the `@prisma/adapter-pg` driver adapter connecting to **Neon PostgreSQL** (serverless). The `proxy()` convention replaces deprecated middleware for auth routing. The design system uses **Tailwind CSS 4** with OKLch color space theming (warm Ivory light + Deep Indigo dark) and full bilingual (Chinese/English) support.
