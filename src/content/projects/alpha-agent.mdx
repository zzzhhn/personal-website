## Background

Quantitative factor research traditionally requires deep domain expertise and tedious manual iteration. Inspired by the KDD '25 AlphaAgent paper, this project automates the full research loop: from natural language hypothesis generation to factor expression synthesis, backtesting, and evaluation — all orchestrated by a multi-agent LLM system.

## Architecture

Four specialized agents collaborate in a feedback loop:

1. **HypothesisAgent** — generates financial hypotheses from natural language queries
2. **FactorAgent** — synthesizes factor expressions in a Qlib-compatible DSL (recursive descent parser with operator precedence)
3. **BacktestAgent** — evaluates factors via IC, ICIR, Sharpe ratio, alpha decay analysis on CSI300 data
4. **EvalAgent** — judges results and provides structured feedback for iterative refinement

## Key Results

- End-to-end pipeline processes "find short-term reversal factors" in 2-3 iterations
- Custom recursive descent parser supports infix math + 20 time-series/cross-section operators
- 168 tests passing with 87% coverage on core modules
- Zero external framework dependencies for agent orchestration (raw HTTP to Ollama)
